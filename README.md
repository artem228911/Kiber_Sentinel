# CYBER SENTINEL PRO

Антивирус **CYBER SENTINEL PRO** — это мощная программа на Python с графическим интерфейсом, созданная для защиты системы и анализа подозрительных файлов. Она объединяет классический антивирусный движок, эвристический анализ и дополнительные утилиты для оптимизации производительности ПК.
